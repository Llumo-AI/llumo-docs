### 1.1 Resources  
**Description:** A link to documentation, tutorials, and guides.  
**Purpose:** Helps users understand the platform's features and functions.  

### 1.2 Need Help?  
**Description:** A support section for troubleshooting and assistance.  
**Purpose:** Connects users with FAQs, customer support, or help resources.  

### 1.3 API Key  
**Description:** Displays the user's API key, partially masked for security.  
**Purpose:** Enables programmatic access to the platform.  
**Action:** The copy icon allows users to copy the key to their clipboard.  

### 1.4 New  
**Description:** A button to create a new workspace or session.  
**Purpose:** Allows users to start a fresh project or task with no preloaded settings or data.  

### 1.5 Playground  
**Description:** Displays the name of the active workspace or playground (e.g., "playground-16").  
**Purpose:** Represents the current working environment, where users can experiment with data, prompts, or models.  

### 1.6 See All  
**Description:** A button to view and manage all existing playgrounds or workspaces.  
**Purpose:** Provides a centralized view to switch between, delete, or organize multiple projects.  

### 1.7 Connect API  
**Description:** A button to integrate the API with third-party applications.  
**Purpose:** Simplifies external system integration.  

### 1.8 Generate Sample  
**Description:** A tool to generate a sample or template for users to explore features.  
**Purpose:** Serves as a starting point for new users.  

### 1.9 Upload  
**Description:** Allows users to upload files for processing or analysis.  
**Purpose:** Enables file-based input for various tasks.  

### 1.10 Evaluate  
**Description:** Opens options for evaluating outputs or performance metrics.  
**Purpose:** Helps users analyze results based on chosen criteria.  

### 1.11 Settings (Gear Icon)  
**Description:** Provides access to user and platform settings.  
**Purpose:** Allows customization of preferences and configurations.  

### 1.12 User Profile  
**Description:** Displays the logged-in user's profile and settings.  
**Purpose:** Offers quick access to account management options.  

### 1.13 More Options (Three Dots Icon)  
**Description:** Opens additional tools or menu options.  
**Purpose:** Expands functionality for advanced users.

**Description:** Adjusts the height of rows displayed in tables or datasets.  

### Levels of Adjustment:  
- **Small:** Minimal row height for compact data display.  
- **Medium:** Default row height for balanced readability and visibility.  
- **Large:** Increased row height for easier readability.  
- **Extra Large:** Maximized row height, suitable for detailed row content.  
- **All Text:** Automatically adjusts row height to fit all text without truncation.  

**Purpose:** Customizes the display to suit user preferences or specific data needs.

## Here, in the above screen, perform the following  

### Selected Checkbox  
Indicates that 100 rows (or data entries) are currently selected for batch processing, such as cloning or evaluation.  

### Clone Button  
Creates a duplicate of the selected rows or data entries. Useful for experimenting with slight modifications while retaining the original data.  

### Delete Button  
Deletes the selected rows or data entries from the evaluation table. This action is typically irreversible, so users need to confirm before proceeding.  

### Run Button  
Executes the evaluation process for the selected rows. This triggers the prompt and output generation along with custom metric evaluations.  

### Close Button  
Exits the current evaluation session or clears the current selection of rows, returning users to the previous state or dashboard.
